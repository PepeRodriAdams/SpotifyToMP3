Nuestra API REST
HTTP Plantilla / URI Comentarios
GET /link

Con este comando GET devuelve un JSON con todos los links de descarga de la aplicación. Si en algún momento estuviese vacía, el sistema devolverá un “404 NOT FOUND” y un mensaje de error.

GET /link/url

Con este comando GET devuelve el link de descarga del video de Youtube con url = url elegida por el usuario. Si no disponemos de ningún link de descarga, el sistema devolverá un “404 NOT FOUND” y un mensaje de error.

POST /link

Con el comando POST se añadirá un link con los datos que se pasen en el cuerpo de la petición en formato JSON Si todo es correcto, lo añadiremos y el sistema devolverá un “201 CREATED”. Si los valores no son válidos u olvida algún dato, el sistema devolverá un “400 BAD REQUEST”.

PUT /link

Con el comando PUT se actualizará el link cuyos datos se pasan en el cuerpo de la petición en formato JSON. Si el link que busca no se encuentra en nuestra aplicación, el sistema devolverá un "404 NOT FOUND".

DELETE /link/url

Con el comando DELETE se eliminará el link del video de Youtube url = url elegida por el usuario. Si el link a eliminar no está en nuestra aplicación, el sistema devolverá un “400 BAD REQUEST”.
